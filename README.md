Lasta Thymeleaf
=======================
Thymeleaf adapter for LastaFlute

# Information
## Maven Dependency in pom.xml
```xml
<dependency>
    <groupId>org.lastaflute.html</groupId>
    <artifactId>lasta-thymeleaf</artifactId>
    <version>0.3.2</version>
</dependency>
```
